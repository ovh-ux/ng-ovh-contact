ovhContactChoice
directive in module ovhContact

Description

This directive allows you selecting a contact from /me/contact list or from a custom list.

It provides possibility to add/edit a contact.

Note : This directive is automatically loaded by its parent (ovhContact).

Dependencies

  • ovhContact

Usage

as element:
<ovh-contact-choice
       [ovh-contact-choice-filter="{Function}"]
       [ovh-contact-choice-custom-list="{Array<OvhContact>}"]
       [ovh-contact-choice-options="{Object}"]>
</ovh-contact-choice>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ovhContactChoiceFilter
(optional)
Function

A filter function that will be called to filter contacts list (for filtering identical contacts for example).

ovhContactChoiceCustomList
(optional)
Array<OvhContact>

A custom list of contact. Using this won't load the contact list and creation/edition won't be possible.

ovhContactChoiceOptions
(optional)
Object

Others options available for selecting a contact.

PropertyTypeDetails
allowCreation
(optional)
Boolean

Allow or not contact creation.

(default: true)

allowEdition
(optional)
Boolean

Allow or not contact creation.

(default: true)

autoCreateContact
(optional)
Boolean

Auto create a contact from connected nic informations if contact list is empty.

(default: true)
