OvhContact
object in module ovhContact

Description

Dependencies

Methods

  • create()

    Create an ovh contact with the current options. Call POST /me/contact/{contactId} API.

    Returns
    Promise

    That returns the current instance of OvhContact created.

  • hasChange(path)

    Check if the contact options have been modified since startEdition has been called.

    Parameters
    ParamTypeDetails
    path
    (optional)
    		String

    Path of the attributes of the contact to check. If not provided, this will check all attributes paths of the instance.

    Returns
    Boolean

    True if an options has changed. False otherwise.

  • save()

    Save the current contact instance options. Call PUT /me/contact/{contactId} API.

    Returns
    Promise

    That returns the current instance of OvhContact.

  • startEdition()

    Start the edition of a contact. Create a copy of current options to allow you to revert when necessary.

    Returns
    OvhContact

    The current instance of OvhContact.

  • stopEdition(cancel)

    Stop the edition of the contact.

    Parameters
    ParamTypeDetails
    cancel
    (optional)
    		Boolean

    Flag telling if we have to reset saved options when startEdition has been called.

    Returns
    OvhContact

    The current instance of OvhContact.

Example

angular.module("myManagerApp").controller("MyTestCtrl", function (OvhContact) {
    var myContact = new OvhContact({
        email: "my-contact@email.com",
        firstName: "my-contact-firstName",
        lastName: "my-contact-lastName"
    });
});
